# Car Showroom Management System

# Overview

A Java OOP-based console application to manage showroom operations including Employees, Customers, Cars, Showrooms, and Sales.

# Features

* Add Showrooms
* Add Employees
* Add Cars
* Add Customers
* Sell Cars
* Search Cars
* Sales Report

# Tech Stack

* Java (OOP)
* Console-based system

# How to Run

1. Compile:
   javac *.java

2. Run:
   java Main

# Project Structure

* Showroom.java
* Employee.java
* Customer.java
* Cars.java
* Vehicle.java
* Person.java
* utility.java
* Main.java
